> 一个 [Scriptable](https://scriptable.app/) 小组件，将js代码复制到scriptable即可运行，可作为桌面小组件。根据子午流注**纳甲法**实时推算当前时辰的开穴，并以色示经、日夜适配，随手一瞥即可知此刻当开何穴。本版在v0.1上修正了部分公式算法bug。

> A Scriptable widget that calculates the currently open acupoint based on the **Na Jia method** of Zi Wu Liu Zhu (Midnight-Noon Ebb-Flow), with Five-Element color coding and day/night theme support. This version revise some bugs from v0.1 formula.

## 背景 · Background

**子午流注**是中医针灸学中按时取穴的古典针法，认为人体气血在十二经脉中的流注随昼夜时辰而有盛衰开阖。**纳甲法**以**日干**为主推算当日各时辰的开穴，按井、荥、输、经、合五输穴逐时辰流注，阳日开阳经，阴日开阴经。

本 Widget 将该理论编码为 JavaScript 算法，通过 Scriptable 在 iOS / macOS 桌面实时展示当前时辰应当开启的穴位。

---

## 功能特性 · Features

- **纳甲法主表** — 按日干 + 时支直接查表开穴
- **返本还原** — 逢输穴时自动追加当日值日经的原穴
- **夫妻互用** — 纳甲法无穴时自动转向合日干支查表
- **辰时气纳三焦 / 血归包络** — 辰时额外开三焦经、心包经合穴
- **补穴规则** — 主表与夫妻互用皆无穴时，按阴阳同异规则推算备选穴位
- **五行配色** — 根据开穴所属经络的五行属性自动切换背景色
- **日夜模式** — 23:00–06:00 自动切换为深色背景
- **信息完整** — 公历时间、日干支、时辰、阴阳属性、开穴名称一屏呈现

### 返本还原

当按纳甲法开到的穴位为**输穴**时，需「返本还原」—— 加开当日值日经的**原穴**。例如甲日胆经值日，遇输穴则加开胆经原穴「丘墟」。

### 夫妻互用

若当前日干在当前时支无纳甲法穴位，则转向**合日干支**（甲↔己、乙↔庚、丙↔辛、丁↔壬、戊↔癸）查表。来源于「刚柔相配，阴阳相合」的合日互用原则。

### 气纳三焦 / 血归包络

五输穴（井→荥→输→经→合）依次开完后，日干重见时——即同一日干再次出现的时辰，气血不再走本经，而是分别归入三焦和心包，补全十天干配十二经脉的对应关系。理论出自明代徐凤《针灸大全》：

> 三焦乃阳气之父，包络乃阴血之母。

**阳日（甲、丙、戊、庚、壬）→ 气纳三焦，开生我穴**：阳经开完五输，日干重见时，按「他生我」原则取三焦经腗穴。例如甲日申时重见甲，甲属木，水生木（他生我），开三焦经荥水穴液门。

**阴日（乙、丁、己、辛、癸）→ 血归包络，开我生穴**：阴经开完五输，日干重见时，按「我生他」原则取心包经腗穴。例如乙日未时重见乙，乙属木，木生火（我生他），开心包经荥火穴劳宫。

### 原理2——按步计算法

### 第1步：天干配脏腑，阳进阴退

判断日干阴阳，确定当日值日经脉：

| 阳日 | 甲(胆) | 丙(小肠) | 戊(胃) | 庚(大肠) | 壬(膀胱) |
|:---:|:---:|:---:|:---:|:---:|:---:|
| **阴日** | 乙(肝) | 丁(心) | 己(脾) | 辛(肺) | 癸(肾) |

### 第2步：阳日开阳经井穴，按五输穴次序开阳时

- 阳日值日经于阳时（子寅辰午申戌）依次开**井 → 荥 → 输 → 经 → 合**
- **返本还原**：开输穴时，同开当日值日经的原穴
- **气纳三焦**：阳经开到合穴后，下一阳时气纳三焦，开生我穴

### 第3步：阴日开阴经井穴，按五输穴次序开阴时

- 阴日值日经于阴时（丑卯巳未酉亥）依次开**井 → 荥 → 输 → 经 → 合**
- **返本还原(以输为原)**：开输穴时，同开井穴所属经脉的原穴
- **血归包络**：阴经开到合穴後，下一阴时血归包络，开我生穴

### 第4步：合日互用 + 阳经阳时 / 阴经阴时补穴

**合日互用（夫妻互用）**：根据天干合化五行（甲己合土、乙庚合金、丙辛合水、丁壬合木、戊癸合火），阳日阴时或阴日阳时，互开对合日的同地支时辰穴位。

**阳经阳时补穴**（阳日值日经，遇阳时闭穴时）：
- 子补井 → 寅补荥 → 辰补输 → 午补经 → 申补合 → 戌补纳

**阴经阴时补穴**（阴日值日经，遇阴时闭穴时）：
- 丑补井 → 卯补荥 → 巳补输 → 未补经 → 酉补合 → 亥补纳

### 第5步：一、四、二、五、三、〇反克取穴法

当1-4步均未找到开穴时，触发反克取穴。按五行逆克顺序重新排列五输穴：
- **一(井) → 四(经) → 二(荥) → 五(合) → 三(输) → 〇(纳/归)**

解决了合日互用后仍存在的24个闭穴时辰，将10日120个时辰全部补全。

### 查找流程2

```
getAcupoint142530(dayGan, shiZhi) {
  1. acupointMantraMap[日干][时支]    → 纳甲法主表
  2. acupointMantraMap[合日干][时支]  → 夫妻互用
  3. getStep4BuXue(日干, 时支)        → 阳经阳时 / 阴经阴时补穴
  4. lookup142530(日干, 时干, 时支)   → 反克取穴
}
```

### 时干推算

时干由日干 + 时支决定：

> 甲己起甲子，乙庚起丙子，丙辛起戊子，丁壬起庚子，戊癸起壬子

当前时干 = 子时干序数 + 时支序数（模10）
