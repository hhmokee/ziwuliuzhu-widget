# 142530取穴法 · 一、四、二、五、三、〇反克取穴

> 子午流注纳甲法补穴体系的第5步（最后兜底）：当纳甲法主表（1-3步）+ 合日互用与阳经阴经补穴（第4步）仍留有闭穴时，以**五行反克**逻辑的 **一、四、二、五、三、〇** 顺序将剩余24个闭穴时辰全部补开。

> 本模块将单玉堂反克取穴表编码为 `fanKeTable`，并实现完整的5步纳甲法取穴流程。

## 纳甲法5步流程

本模块严格遵循子午流注纳甲法的5步递进逻辑，依次尝试开穴：

### 第1步：天干配脏腑，阳进阴退

判断日干阴阳，确定当日值日经脉：

| 阳日 | 甲(胆) | 丙(小肠) | 戊(胃) | 庚(大肠) | 壬(膀胱) |
|:---:|:---:|:---:|:---:|:---:|:---:|
| **阴日** | 乙(肝) | 丁(心) | 己(脾) | 辛(肺) | 癸(肾) |

### 第2步：阳日开阳经井穴，按五输穴次序开阳时

- 阳日值日经于阳时（子寅辰午申戌）依次开**井 → 荥 → 输 → 经 → 合**
- **返本还原**：开输穴时，同开当日值日经的原穴
- **气纳三焦**：阳经开到合穴后，下一阳时气纳三焦，开生我穴

### 第3步：阴日开阴经井穴，按五输穴次序开阴时

- 阴日值日经于阴时（丑卯巳未酉亥）依次开**井 → 荥 → 输 → 经 → 合**
- **返本还原(以输为原)**：开输穴时，同开井穴所属经脉的原穴
- **血归包络**：阴经开到合穴後，下一阴时血归包络，开我生穴

### 第4步：合日互用 + 阳经阳时 / 阴经阴时补穴

**合日互用（夫妻互用）**：根据天干合化五行（甲己合土、乙庚合金、丙辛合水、丁壬合木、戊癸合火），阳日阴时或阴日阳时，互开对合日的同地支时辰穴位。

**阳经阳时补穴**（阳日值日经，遇阳时闭穴时）：
- 子补井 → 寅补荥 → 辰补输 → 午补经 → 申补合 → 戌补纳

**阴经阴时补穴**（阴日值日经，遇阴时闭穴时）：
- 丑补井 → 卯补荥 → 巳补输 → 未补经 → 酉补合 → 亥补纳

### 第5步：一、四、二、五、三、〇反克取穴法（最后兜底）

当1-4步均未找到开穴时，触发反克取穴。按五行逆克顺序重新排列五输穴：
- **一(井) → 四(经) → 二(荥) → 五(合) → 三(输) → 〇(纳/归)**

解决了合日互用后仍存在的24个闭穴时辰，将10日120个时辰全部补全。

## 背景

子午流注纳甲法按日干推算开穴，阳日阳时开阳经穴、阴日阴时开阴经穴。以徐凤《针灸大全》所传系统为准（仅十经参与流注），每日仅6个时辰有穴可开：

- 纳甲法主表（1-3步）：开60穴 / 120时辰
- 合日互用（第4步前半）：另开36穴，累计96穴
- 阳经阴经补穴（第4步后半）：补开部分剩余闭穴

但仍有**24个时辰**处于闭穴状态。

现代针灸家**单玉堂**（1902-1983，北京中医学院元老）在其《子午流注与灵龟八法讲稿》中系统提出了**一、四、二、五、三、〇反克取穴法**，以五行逆克逻辑重排五输穴，作为**第5步兜底**，将最后24个闭穴时辰全部补开。

## 功能特性

- **5步完整流程** — 纳甲法主表 → 夫妻互用 → 合日补穴 → 反克取穴
- **反克表硬编码** — 60条（10时干 x 6反克位）直接查表，零推算误差
- **五输类型标注** — 每条记录标明井 / 经 / 荥 / 合 / 输 / 纳
- **即插即用** — 替换原有 `getAcupoint` 函数即可
- **独立查表函数** — `lookup142530(dayGan, shiGan, shiZhi)` 无外部依赖
- **模块化导出** — CommonJS 可引用

## 技术原理

### 五行反克顺序

反克取穴法按五行逆克顺序排列五输穴：

```
常规顺序：井 (1) → 荥 (2) → 输 (3) → 经 (4) → 合 (5)
反克顺序：井 (1) → 经 (4) → 荥 (2) → 合 (5) → 输 (3) → 纳/归 (0)
```

逆克链条：经金克井木 → 荥火克经金 → 合水克荥火 → 输土克合水 → 纳/归终转

### 完整查找流程

```
getAcupoint142530(dayGan, shiZhi) {
  1. acupointMantraMap[日干][时支]    → 纳甲法主表
  2. acupointMantraMap[合日干][时支]  → 夫妻互用
  3. getStep4BuXue(日干, 时支)        → 阳经阳时 / 阴经阴时补穴
  4. lookup142530(日干, 时干, 时支)   → 反克取穴（最后兜底）
}
```

### 时干推算

时干由日干 + 时支决定：

> 甲己起甲子，乙庚起丙子，丙辛起戊子，丁壬起庚子，戊癸起壬子

当前时干 = 子时干序数 + 时支序数（模10）
